💡 Recommendation: For production use, we recommend deploying a self-hosted OSRM service. This ensures data privacy, reliability, and optimal performance for large-scale distance calculations.

Step 1: Install Docker

  1. Download Docker Desktop for Mac
  2. Run the installer and follow the on-screen instructions
  3. Open Terminal and verify: docker --version
⏱️ Time: 10-15 minutes

Step 2: Download OSRM Data

Get India data from Geofabrik:

curl -o india.osm.pbf https://download.geofabrik.de/asia/india-latest.osm.pbf
⏱️ Time: 30-60 minutes (data is ~900MB)

Step 3: Build OSRM Docker Image

docker run -t osrm/osrm-backend osrm-extract -p /opt/car.lua /data/india.osm.pbf
⏱️ Time: 15-30 minutes

Step 4: Start OSRM Server

docker run -d -p 5000:5000 -v $(pwd):/data osrm/osrm-backend osrm-routed --algorithm mld /data/india.osm.pbf
⏱️ Time: 5 minutes

Step 5: Verify Installation

Test the API:

curl "http://localhost:5000/route/v1/driving/77.5946,12.9716;77.5900,12.9700?overview=false"

Step 1: Install Docker Desktop

  1. Download Docker Desktop for Windows
  2. Run the installer and follow the on-screen instructions
  3. Enable WSL 2 (Windows Subsystem for Linux 2) when prompted
  4. Open PowerShell and verify: docker --version
⏱️ Time: 15-20 minutes

Step 2: Download OSRM Data

Open PowerShell and download India data:

Invoke-WebRequest -Uri "https://download.geofabrik.de/asia/india-latest.osm.pbf" -OutFile "india.osm.pbf"
⏱️ Time: 30-60 minutes

Step 3: Create Docker Container

docker run -t osrm/osrm-backend osrm-extract -p /opt/car.lua /data/india.osm.pbf
⏱️ Time: 20-40 minutes

Step 4: Start OSRM Service

docker run -d -p 5000:5000 -v ${pwd}:/data osrm/osrm-backend osrm-routed --algorithm mld /data/india.osm.pbf
⏱️ Time: 5 minutes

Step 5: Test Your Setup

Invoke-WebRequest "http://localhost:5000/route/v1/driving/77.5946,12.9716;77.5900,12.9700?overview=false"

Step 1: Install Docker

sudo apt-get update && sudo apt-get install docker.io docker-compose
  1. Verify installation: docker --version
  2. Add user to docker group: sudo usermod -aG docker $USER
⏱️ Time: 10 minutes

Step 2: Download OSRM Data

wget https://download.geofabrik.de/asia/india-latest.osm.pbf
⏱️ Time: 30-60 minutes

Step 3: Create OSRM Data Files

docker run -t -v $(pwd):/data osrm/osrm-backend osrm-extract -p /opt/car.lua /data/india.osm.pbf
docker run -t -v $(pwd):/data osrm/osrm-backend osrm-partition /data/india.osm
docker run -t -v $(pwd):/data osrm/osrm-backend osrm-customize /data/india.osm
⏱️ Time: 30-60 minutes

Step 4: Start OSRM Server

docker run -d -p 5000:5000 -v $(pwd):/data osrm/osrm-backend osrm-routed --algorithm mld /data/india.osm
⏱️ Time: 5 minutes

Step 5: Verify Installation

curl "http://localhost:5000/route/v1/driving/77.5946,12.9716;77.5900,12.9700?overview=false"

Team Sharing & Collaboration

Once you have OSRM running, share the server URL with your team:

http://[your-server-ip]:5000
  • For local network: use your machine's IP address (find with ipconfig getifaddr en0 on Mac or hostname -I on Linux)
  • For remote access: configure firewall rules and consider using a reverse proxy (nginx, Apache)
  • For production: use SSL/TLS certificates and authentication
